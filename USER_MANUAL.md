### 1. 初次见面：如何开始

你收到的应该是一个名为 `DreamWeaver` 的文件（在 Windows 上可能是 `DreamWeaver.exe`）。这就是我们的主程序。

请将这个文件放在一个你喜欢的文件夹里，比如桌面上的一个新文件夹。

**第一次运行**：
直接双击 `DreamWeaver` 文件。程序会启动，并可能会弹出一个黑色的命令行窗口（请不要关闭它）。同时，它会自动在你的默认浏览器中打开一个网页。

最重要的是，在 `DreamWeaver` 文件所在的文件夹中，你会发现一个新生成的文件，名为 `config.ini`。这是我们进行下一步的关键。

---

### 2. 第一步：配置你的专属“钥匙” (API Key)

为了让“织梦者”能够连接到强大的人工智能服务来为你生成图片和文字，你需要提供一个专属的“钥匙”，也就是 **API Key**。

#### 什么是 API Key？

你可以把它想象成一个网站的“密码”。你向服务商（比如豆包）申请一个 API Key，就等于获得了一个使用他们AI服务的许可。

#### 如何配置？

1.  **找到 `config.ini` 文件**：它就在你存放 `DreamWeaver` 程序的那个文件夹里。

2.  **用文本编辑器打开它**：
    *   **Windows 用户**：右键点击 `config.ini` -> 选择“打开方式” -> 选择“记事本”。
    *   **Mac 用户**：右键点击 `config.ini` -> 选择“打开方式” -> 选择“文本编辑”。

3.  **填写你的 API Key**：
    打开后，你会看到如下内容：

    ```ini
    [api_keys]
    google =
    qwen =
    doubao =
    ```

    你需要将你从服务商那里获取到的 API Key，**直接粘贴在等号的后面**。

    **例如**，假设你的 Google API Key 是 `AIzaSy...`，你的 Qwen API Key 是 `sk-1234...`，那么你应该这样填写：

    ```ini
    [api_keys]
    google = AIzaSy...
    qwen = sk-1234...
    doubao =
    ```

    **重要提示**：
    *   你**不需要**填写所有三个 API Key。你只需要填写你拥有的、想要使用的那个或那几个即可。
    *   **不要**在 Key 的前后添加引号或其他任何符号。
    *   修改完成后，请**保存**并关闭 `config.ini` 文件。

### 3. 开始织梦：如何使用本应用

配置好 API Key 后，你就可以正式开始使用了！

1.  **运行程序**：再次双击 `DreamWeaver` 文件。程序会自动打开浏览器页面。

2.  **进入“梦想画布”**：在首页上，点击“开始你的梦想”按钮。

3.  **描绘你的梦想**：
    *   **上传照片**：点击图片区域，选择一张你自己的、清晰的正面照片。
    *   **选择 API 服务**：在下拉菜单中，选择一个你已经配置好 Key 的服务。如果你只配置了一个，这里就只会显示一个选项。
    *   **你的名字**：输入你的名字或昵称。
    *   **描述你的梦想**：用文字详细描述你的梦想，越具体越好。例如：“成为一名宇航员，在月球上插上旗帜”、“开一家温馨的咖啡馆，每天为客人制作手冲咖啡”。

4.  **生成梦想**：点击“可视化我的梦想”按钮。程序需要一些时间来为你创作，请耐心等待。

### 4. 保存与分享你的梦想

当AI为你生成好未来的图景后，你会看到一张为你量身定做的图片和一段激励人心的话语。

在页面的下方，有三个按钮供你使用：

*   **二维码分享**：点击后会生成一个二维码。其他人用手机扫描这个二维码，就可以看到你的梦想图景。
*   **下载梦想图**：点击后会直接下载由AI为你生成的、代表你梦想成真的那张**单图**。
*   **下载合成图**：点击后会下载一张**组合图片**，这张图片包含了你的梦想图、激励文字和你的名字，非常适合在社交媒体上分享。

### 5. 常见问题解答 (FAQ)

*   **问：我双击程序后，为什么没有反应？**
    *   答：请确保你没有误删任何文件。另外，可以尝试右键点击程序，选择“以管理员身份运行”。

*   **问：页面提示“未找到可用的 API 服务”怎么办？**
    *   答：这说明你没有正确配置 `config.ini` 文件。请回到[第二步](#2-第一步配置你的专属钥匙-api-key)，确保你已经将至少一个 API Key 填写正确并保存了文件。

*   **问：生成图片时出错了怎么办？**
    *   答：请检查：
        1.  你的 API Key 是否正确，或者是否已过期/欠费。
        2.  你的网络连接是否正常。
        3.  可以尝试更换一个 API 服务商再试一次。

*   **问：我需要一直开着那个黑色的命令行窗口吗？**
    *   答：是的。那个窗口是程序的“心脏”，关闭它就会关闭整个应用。当你不想使用时，可以直接关闭那个窗口。
